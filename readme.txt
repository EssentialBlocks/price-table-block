= Modern Way: =

3. Install in 1-click

= Old Way: =
1. Upload `price-table-block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the [Documentation](https://essential-blocks.com/docs/)
